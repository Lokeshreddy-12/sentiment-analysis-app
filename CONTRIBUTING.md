# Contributing

Thanks for your interest in contributing to this project! A few ways to help:

- Report issues or suggest features via GitHub Issues
- Open pull requests for bug fixes, docs improvements, or enhancements
- Keep changes small and focused, and include tests where appropriate

Before opening a PR:
- Ensure your changes are linted and tests pass
- Add clear descriptions to your PR and link any relevant issue
